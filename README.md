# JunkBox_Lib
C/C++ Static Library

## Compile
```
git clone https://github.com/JunkBox-Library/JunkBox_Lib.git
cd JunkBox_Lib
./config.sh
./configure --enable-jpeg --enable-openjpeg=2.5 --enable-ssl --enable-ldap   # Ex.
make
cd C++Lib
make
cd ../..
```
